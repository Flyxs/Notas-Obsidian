# Decodificadores

Um decodificador é um circuito de múltiplas entradas e múltiplas saídas que converte códigos de entrada em códigos de saída, onde os dois códigos são diferentes.

Em geral, o código de entrada tem menos bits que o código de saída, e existe um mapeamento um-pra-um do código de entrada para o código de saída.

O código de saída mais utilizado é o 1-de-m, que usa m bits para m códigos e, para código, apenas um dos m bits ativos. Por exemplo:

| Ativo em alto | Ativo em baixo |
| :------------ | -------------- |
| 1000          | 0111           |
| 0100          | 1011           |
| 0010          | 1101           |
| 0001          | 1110           |
